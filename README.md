## TERMUX USER
```bash
> pkg update && pkg upgrade
> pkg install nodejs
> pkg install libwebp
> pkg install git
> pkg install ffmpeg
> pkg install yarn
> termux-setup-storage
> git clone https://github.com/Nurutomo26/wa-md.git
> cd wa-md
> yarn
> npm start
```
